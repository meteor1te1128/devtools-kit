How to use the JSON Formatter

Paste or type your JSON into the Input panel, then click Format / Beautify to get a nicely indented, human-readable version. You can customise the indent size (2 spaces, 4 spaces, or tabs) and optionally sort keys alphabetically before formatting.

The Minify button removes all whitespace to produce the most compact JSON possible — ideal for reducing payload sizes in APIs. The Validate button checks your JSON without modifying it, and points to the exact line and column of any syntax error.

Everything runs entirely in your browser using the native JSON.parse API — your data is never uploaded to any server.
